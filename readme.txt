# Daily Bread WordPress Plugin

## Description

Daily Bread is a simple WordPress plugin that displays a random proverb on the admin screen. The proverbs are loaded from a JSON file.

## Installation

1. Upload the `daily-bread` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

## Usage

Once activated, the plugin will automatically display a random proverb on the admin screen. The proverbs are loaded from a `proverbs.json` file located in the plugin directory.
